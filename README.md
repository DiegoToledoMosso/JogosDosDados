# Jogo Do Dado

![](https://imgur.com/QaGvTOp.gif)

## Introdução

O jogador e o computador competem em uma corrida virtual. 
Cada um rola um dado virtual (geração de números aleatórios) 
para avançar em uma pista. O primeiro a alcançar ou ultrapassar a linha de chegada vence.

## Funcionalidades

- Controle da posição do jogador
- Geração de números aleatórios para simular um dado (1-6)
- Exibir o resultado do lançamento do dado
- Permitir que o usuário pressione Enter para lançar o dado
- Armazenar a posição do jogador e computador na pista e atualizar o valor após o lançamento do dado
- Exibir a posição atual do jogador e do computador na pista
- Definir a linha de chegada em 30 verificar se o jogador alcançou ou ultrapassou a linha de chegada
- Permitir o jogador e o computador a realizar várias jogadas
- Informar que o computador e o jogador estão jogando
- Atualizar a posição do jogador e do computador após seu lançamento de dado
- Exibir mensagens informativas quando os eventos ocorrerem
- Atualizar a posição do jogador e do computador conforme as regras dos eventos
- Verificar se o jogador ou o computador alcançou ou ultrapassou a linha de chegada
- Informar quem venceu o jogo
- Implementar turnos alternados entre jogador e computador
